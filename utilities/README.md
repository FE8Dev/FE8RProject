# Utilities

Tools to help with LT editing

## Chapter Unit Batch Editor

1. Install FreeSimpleGUI via pip.

    ```bash
    python -m pip install FreeSimpleGUI
    ```

2. Run using python. To run it in the background, add `&` to the end of the command on macOS/Linux (bash) or, run with `pythonw` instead of `python` on Windows.

    ```bash
    python level_unit_batch_edit.py
    python level_unit_batch_edit.py &
    pythonw level_unit_batch_edit.py

    ```
